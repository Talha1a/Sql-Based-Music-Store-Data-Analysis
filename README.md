# Sql-Based-Music-Store-Data-Analysis
This project covers in depth EDA of music store data and identifies useful patterns and answers.

SQL project to analyze online music store data

This project tells that how to analyze the music playlist database. You can examine the dataset with SQL and help the store understand its business growth by answering simple questions.

## Database and Tools
* Postgre SQL
* PgAdmin4
